# 当前 USART1 协议状态

## 文档范围

当前接口是 STM32F407 底盘电机控制器的 `USART1`，波特率为 `115200`。
正式 MOWEN 二进制协议保持不变。本文件说明当前正式协议、保留的 ASCII
调试协议，以及最近一次调参测试后的设备状态。

## 正式协议：MOWEN

正式运动命令使用固定长度的 12 字节二进制帧：

```text
AA BB 0A 12 02 XL XH YL YH ZL ZH 00
```

其中 `X`、`Y`、`Z` 是小端序有符号 `int16` 数据，分别表示 `vx`（米/秒）、
`vy`（米/秒）和 `wz`（弧度/秒），实际发送值为物理量乘以 `1000`。最后一个
字节是保留字节，必须保持为 `00`。

运动命令建议每 `20~100 ms` 刷新一次。固件通信看门狗超时时间为 `200 ms`；
超过该时间没有收到有效运动命令时，固件会清零全部目标并关闭 PWM 输出。

正式协议反馈帧仍为 12 字节：

```text
AA BB 0A 12 VX VX VY VY WZ WZ 00 CHECKSUM
```

校验和为第 2 至第 9 字节之和的低 8 位：

```text
CHECKSUM = sum(frame[2]..frame[9]) & 0xFF
```

## ASCII 调试协议

使用以下 sideband 帧切换到 ASCII 调试模式：

```text
FE EF 50 00 50 FD
```

ASCII 命令以 `CR/LF` 结尾，常用命令如下：

```text
SET,<参数编号>,<参数值>
MSET,<电机编号>,<参数编号>,<参数值>
COMMIT
RESTORE
ZERO
```

`SET,0,<参数值>` 用于在线修改轮子半径。修改成功后，若底盘正在运动，
固件会使用最新参数重新计算最近一次速度命令，因此不需要重新发送
`VEL` 命令即可生效。

底盘参数编号如下：

| 编号 | 参数 | 单位/取值 |
| ---: | --- | --- |
| 0 | 轮子半径 `wheel_radius` | m，`0.01~0.30` |
| 1 | 轮距 `track_width` | m，`0.05~1.20` |
| 2 | 轴距 `wheel_base` | m，麦克纳姆模式必须大于 0 |
| 3 | 减速比 `reduction_ratio` | `0.10~500.0` |
| 4 | 编码器 PPR `encoder_ppr` | `1~65535`，写入时取整 |
| 5 | 电机 0 方向符号 | `+1` 或 `-1` |
| 6 | 电机 1 方向符号 | `+1` 或 `-1` |
| 7 | 电机 2 方向符号 | `+1` 或 `-1` |
| 8 | 电机 3 方向符号 | `+1` 或 `-1` |
| 9 | 最大线速度 `max_linear_speed` | m/s，`0~5.0` |
| 10 | 最大角速度 `max_angular_speed` | rad/s，`0~20.0` |
| 11 | 最大线加速度 `max_linear_accel` | m/s2，`0~20.0` |
| 12 | 最大角加速度 `max_angular_accel` | rad/s2，`0~50.0` |
| 13 | PWM 限制 `pwm_limit` | `1~65535`，写入时取整 |

例如，将当前 profile 的轮子半径修改为 `0.040 m`：

```text
SET,0,0.040
```

电机 PID 和前馈参数使用 `MSET`，编号对所有电机相同：

| 参数编号 | 参数 | 单位/说明 |
| ---: | --- | --- |
| 0 | `kp` | 比例增益 |
| 1 | `ki` | 积分增益 |
| 2 | `kd` | 微分增益 |
| 3 | `kv` | 速度前馈增益 |
| 4 | `k_static` | 静态摩擦前馈 |
| 5 | `output_limit` | PWM 输出上限 |
| 6 | `integral_limit` | 积分限幅 |
| 7 | `deadband_rpm` | RPM 死区 |

例如，将电机 2 的 `kp` 修改为 `110`：

```text
MSET,2,0,110
```

`COMMIT` 提交当前 RAM profile 并清除 dirty 标志。`RESTORE` 将当前 profile
恢复为编译时默认值，并停止底盘运动。`ZERO` 清零里程计、运动目标和输出。

ASCII 模式下，固件每 `100 ms` 回传调试状态，包括：

- `TUNE`：当前底盘 profile 和参数状态；
- `STAT`：工作模式、位置和速度状态；
- `IK_RPM`：四个轮子的逆运动学目标 RPM；
- `M0`~`M3`：各电机目标、反馈、PWM 及 PID 参数。

这些回传内容仅属于 ASCII 调试协议，不属于正式 MOWEN 协议。

## 已验证的默认状态

最近一次测试完成后，设备依次执行了 `RESTORE`、`ZERO`，然后切换回 MOWEN
正式协议。当前麦克纳姆底盘默认参数为：

```text
轮子半径 = 0.03500 m
轮距     = 0.25000 m
轴距     = 0.22000 m
减速比   = 30.000
编码器 PPR = 500
方向符号 = [+1, +1, +1, +1]
PWM 限制 = 13999
```

最终在 `COM16` 上验证到 MOWEN 二进制反馈帧，并确认不再收到 ASCII 调试
反馈，说明设备当前处于默认参数和正式协议模式。
